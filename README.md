# The Besserwisser Effect

**Author:** Hakvin Vosteen
**Year:** 2026

## Overview

This repository contains the paper and reproduction code for *The Besserwisser Effect* — an investigation into the structural asymmetry between knowledge consumption and knowledge production, and how this asymmetry shapes learning trajectories, expertise signaling, and epistemic overconfidence.

The paper introduces the **Besserwisser Effect**: the phenomenon whereby complexity growth in a domain outpaces the information integration rate of observers who consume but do not produce knowledge — leading to a widening gap between perceived and actual understanding.

## Reproducing the Figures

**Requirements:** Python 3.x with `matplotlib`, `numpy`, and `scipy`.

```bash
pip install matplotlib numpy scipy
python besserwisser_fig1.py
python besserwisser_fig2.py
```

Each script saves the corresponding figure as a high-resolution PNG (300 dpi).
